This a web application to manage an Apartment. There are 3 roles in this application. Caretaker, Tenant and Admin.

## Core Features:

i. Service Request

ii. Staff Management

iii. Rent Management

iv. In-Out Tracker

v. Online Message and Offline Text Messaging

## Built With

* Bootstrap
* PHP
* MySQL
* Twilio
